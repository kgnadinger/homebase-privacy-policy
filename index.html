HomeBase – Privacy Policy

Last Updated: 11/21/2025

This Privacy Policy explains how HomeBase (“HomeBase,” “the App,” “we,” “our,” or “us”) collects, uses, discloses, and protects personal information. By using HomeBase, you agree to the terms described here.

1. Summary

HomeBase is a family household-management tool used to assign chores, track rewards, and manage parent and child profiles. Data is stored primarily in your personal iCloud account via Apple CloudKit. We do not sell personal data or use it for advertising.

2. Information We Collect

2.1 Information You Provide

  • Household and profile information
  • Parent and child passcodes
  • Chore details, schedules, and configurations
  • Reward point settings
  • Optional chore completion photos
  • Sign in with Apple identity data

2.2 Automatically Collected Information

  • Device model and OS version
  • App version
  • Crash logs and performance data
  • CloudKit metadata generated by Apple

3. How We Use Information

  • Enable household management features
  • Sync data using CloudKit
  • Authenticate admin users
  • Process chore completion and reward points
  • Improve app performance

4. Data Storage

4.1 iCloud CloudKit

Most data is stored in your private iCloud database. We cannot access this data directly.

4.2 Future Optional Backend

If HomeBase introduces optional backend syncing, you will be notified before data is stored on HomeBase servers.
